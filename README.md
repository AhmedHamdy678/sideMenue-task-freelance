## ğŸ“– Overview

The **Smart Accessibility Tool** is a comprehensive web interface designed to enhance web accessibility for users with diverse needs. It provides a robust sidebar assistant that allows users to customize their browsing experience, making web content more accessible to individuals with visual impairments, learning disabilities (like Dyslexia or ADHD), and other accessibility requirements.

This project focuses on providing equitable access to information through a user-friendly, customizable interface supporting both Arabic and English languages.

## âœ¨ Features

### ğŸ§  Accessibility Profiles
Pre-configured profiles designed for specific needs:
- **Epilepsy Safe Mode**: Reduces vivid colors and removes flashing animations.
- **Visually Impaired Mode**: Enhances website visuals for better clarity.
- **Cognitive Disability Mode**: Helps focus on specific content.
- **Motor Impaired Mode**: Optimizes for keyboard navigation.
- **Blindness Mode**: optimized for screen readers.
- **Dyslexia Friendly**: Uses specialized fonts to improve readability.
- **ADHD Friendly**: Reduces distractions to improve focus.

- **Content Scaling**: Adjust content size.

- **Font Adjustments**: Toggle between easy-to-read and dyslexia-friendly fonts.
- **Highlighting**: Emphasize titles and links.
- **Spacing Control**: Adjust line height, letter spacing, and word spacing.
- **Text Alignment**: Control text alignment (left, justify, etc.).
- **Cursor Customization**: High-contrast, big light/dark cursors.
